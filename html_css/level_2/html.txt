* Flexbox

   display: flex;

   All the contenido lo convierte columnas, para evitar eso debes atrapar
   lo que quieres en un div tag. todo lo que quieras ajustar con flex.
   Cada elemento dentro lo trata como una columna.

* Utility class

   Is a class that every time you call the class they will have the CSS
   value propertys.

   class="value value2 value3"
   Cada value dentro de la case es el elemento y aplica el CSS a ese grupo.
